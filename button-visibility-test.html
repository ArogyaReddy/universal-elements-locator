CSS Specificity Explanation

Problem: The generic .pagination button selector was overriding the specific .pagination-mode-btn styles.

Solution: Used .pagination .pagination-mode-btn with !important to ensure proper styling.

/* OLD - Lower specificity */
.pagination-mode-btn { background: green; }
.pagination button { background: white; } ‚Üê This won

/* NEW - Higher specificity */
.pagination .pagination-mode-btn { background: green !important; } ‚Üê This wins
.pagination button { background: white; }
